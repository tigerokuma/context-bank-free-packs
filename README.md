# Context Bank Free Packs

Choose your language:

- [English](README.en.md)

This repository is the public free-pack intake for Context Bank.

- Free packs are reviewed through pull requests in this repository.
- Merge is the approval event.
- After merge, the private marketplace app can ingest approved packs with `pnpm free-pack:sync`.
