# Japan Stock Trends Sample

## Overview

This sample pack is a ZIP-before-source mock for the submit flow.
It shows the expected Context Bank pack layout with a small investing-focused set of files.

## Included Files

- `SKILL.md` for pack metadata and the marketplace preview body
- `knowledge.md` for analysis heuristics
- `data.json` for small structured reference data
- `examples/` for sample requests and expected outputs
- `prompts/` for reusable drafting prompts
- `assets/preview.svg` for optional supporting media

## Usage Notes

Use this pack when you want short, structured notes about:

- recent price direction
- simple moving-average context
- sector-relative strength
- concise briefing output for a teammate

Keep outputs short, concrete, and readable by a non-specialist teammate.

## Review Guidance

Prefer:

- clear trend language
- basic risk framing
- comparison against a small peer set
- bullet summaries before longer explanation

Avoid overstating certainty or presenting forecasts as facts.

## Disclaimer

This pack is for informational use only.
It does not provide financial advice, does not guarantee returns, and should support human judgment rather than replace it.
